# Risks

RISK-001: Electrical education content can be misread as permission to perform hazardous work.
Mitigation: conservative topic categories, repeated disclaimers, pro-only framing, and call-pro notes.

RISK-002: YouTube search results can change and may include unsafe advice.
Mitigation: label search links clearly, prefer conceptual searches, and remind users to follow licensed guidance and code.

RISK-003: Local progress can be lost if the app is uninstalled or storage is cleared.
Mitigation: document local-only limitation and avoid collecting personal data.

RISK-004: EAS Update cannot deliver native config changes.
Mitigation: document when a new APK build is required.

RISK-005: Dependency vulnerabilities can emerge over time.
Mitigation: add audit scripts and document review workflow.
